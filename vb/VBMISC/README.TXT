Ace E's VB Add-ons v1
***
These are some Visual Basic add-ons I created for my own use.  Below is a description of each new add-on.  As new versions of Add-ons comes out, the additions will be documented in this file.

clsRange.cls

For those who programmed Turbo Pascal and loved their functionality with numeric ranges, this module gives a programmer some of that functionality.

DigiChar.ocx

Need a microware display?  This displays all numbers 0-9 and most alphabetic letters perfectly in an LED style display.  Please excuse the cheezyness of some letters.
